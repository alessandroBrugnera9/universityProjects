O projeto foi feito em Django, porque o Ruby on Rails era uma ferramenta que já está perdendo usuários, e o Django está em crescimento. Além disso, no Ruby, muito tempo é perdido com a incompatibilidade - mesmo no Ubuntu.
Como o Django segue as mesmas diretrizes do ROR (DRY, MVC e MVT, e banco de dados relacionais), criei o código em Django - pois pude aprender também do zero um framework que vou usar.

O projeto em django comporta as propastas da ementa da discplina:
 - Visão geral da indústria
 - software
 - Conceituação de metodologias de desenvolvimento de software; metodologias ágeis.
 - Engenharia de Requisitos. Métricas de Software.
 - Arquiteturas para Sistemas de Informação.
 - Noções de práticas profissionais para implementação de software.
 - Engenharia de Qualidade e Testes.
 - Introdução ao Modelo Relacional.
 - Linguagem SQL, Modelagem de Dados ,Construção de sistemas de informação usando arquitetura web.
